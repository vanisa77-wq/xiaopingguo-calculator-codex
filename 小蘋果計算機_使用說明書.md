# 小蘋果計算機 使用說明書

## 一、基本計算

1. 依序按數字與運算符號。
2. 按 `=` 顯示答案。
3. 計算順序為先乘除、後加減。
4. 每次完成的算式與答案會自動保留在上方紙帶。

範例：

`2 + 3 × 4 = 14`

## 二、數字與運算按鍵

- `00`：輸入兩個零。
- `0`：輸入零。
- `.`：輸入小數點。
- `+`：加法。
- `−`：減法。
- `×`：乘法。
- `÷`：除法。
- `=`：計算並顯示答案。

## 三、括號

- 按 `( )` 可智慧加入左括號或右括號。
- 計算機會依目前算式判斷應加入哪一種括號。
- 如果算式缺少開頭左括號，計算時會自動補齊。

範例：

`(2 + 3) × 4 = 20`

## 四、刪除與清除

### 刪除鍵

- 單按 `⌫`：刪除最後一個字。
- 長按 `⌫`：連續刪除。

### AC

- 清除目前輸入的算式。
- 不會刪除上方紙帶紀錄。
- 會結束目前的稅金或發票計算狀態。

## 五、紙帶紀錄

- 每題按 `=` 後，算式與答案會自動加入紙帶。
- 紙帶紀錄採單行、靠左排列。
- 可在顯示區向上滑動，查看先前所有紀錄。
- 一般計算不會顯示「稅金」或「發票」標籤。
- 只有按「5%稅」或「發票」時，才會出現相應標籤。

## 六、ANS 三種答案

三顆 ANS 都使用上一題的答案開始新算式。

### ANS 0

四捨五入至整數。

- `26666.44 → 26666`
- `26666.57 → 26667`

### ANS .00

四捨五入至小數第 2 位。

- `26666.576 → 26666.58`

### ANS ↑

無條件進位至整數。

- `26666.01 → 26667`

按任一 ANS 後仍會保留「複製」鍵，可複製目前顯示的版本。

## 七、設𝒳 記憶數字

`設𝒳` 可把剛算完的答案存成 𝒳，之後在新算式中直接使用，不需要重打很多位數。

### 設定 𝒳

1. 先完成一題計算。
2. 按 `設𝒳`。
3. 計算機會記住目前答案為 𝒳。

範例：

`1 + 456788 = 456789`

按 `設𝒳` 後：

`𝒳 = 456789`

### 使用 𝒳

設定完成後，在下一題算式中按 `設𝒳`，就會插入 𝒳。

範例：

`123 × 𝒳 = 56,185,047`

也就是：

`123 × 456789 = 56,185,047`

若尚未設定 𝒳，按 `設𝒳` 會提示先算出答案再設定。

## 八、複製答案

1. 按 `=` 完成計算。
2. 顯示區會出現「複製」鍵。
3. 按「複製」即可將目前答案存入手機剪貼簿。
4. 可到 LINE、備忘錄或其他 App 貼上。

按「複製」也會把該答案記錄為可供稅金或發票功能使用的本金。

按「截圖」會自動製作一張只包含完整算式與答案的莫蘭迪風圖片，並開啟 iPhone 分享選單。

## 九、5% 稅金

稅金功能支援兩種本金來源。

### 方法一：直接輸入本金

1. 輸入本金，例如 `58961`。
2. 直接按「5%稅」。
3. 紙帶與顯示區會列出：

`58,961 × 5% = 2,948`

### 方法二：使用複製的本金

1. 先完成一題計算。
2. 按「複製」記錄本金。
3. 按「5%稅」。

稅金依發票使用方式四捨五入至整數，不保留小數。

- `58,961 × 5% = 2,948.05 → 2,948`
- `58,970 × 5% = 2,948.5 → 2,949`

完成後會顯示「稅金」標籤與「複製」鍵。

## 十、發票總額

發票總額為：

`本金 + 5%稅金`

### 方法一：直接輸入本金

1. 輸入本金，例如 `58961`。
2. 直接按「發票」。
3. 顯示：

`58,961 + 2,948 = 61,909`

### 方法二：先算稅金，再算發票

1. 輸入本金。
2. 按「5%稅」。
3. 再按「發票」。
4. 計算機會記住原本金，不會把稅金誤認為本金。

### 方法三：使用複製的本金

1. 完成一題計算。
2. 按「複製」。
3. 按「發票」。

完成後，紙帶及顯示區會出現「發票」標籤，並保留「複製」鍵。

## 十一、結束稅金或發票模式

- 完成稅金或發票計算後，直接按新的數字鍵即可開始一般計算。
- 新算式不會沿用「稅金」或「發票」標籤。
- 按 AC、刪除、括號或一般運算符號，也會結束上一筆特殊計算的顯示狀態。

## 十二、常見提示

- 「算式還沒完成」：最後一個字是運算符號或左括號，請先完成算式。
- 「請直接輸入金額，或先按複製」：尚未提供可計算稅金或發票的本金。
- 「先算一題才有 ANS」：尚未產生上一題答案。
- 「先算出答案，再按設𝒳」：尚未設定 𝒳。
- 除數不可為零。

## 十三、安裝到手機

### iPhone / iPad

1. 使用 Safari 開啟計算機網址。
2. 點「分享」。
3. 選擇「加入主畫面」。
4. 名稱輸入「小蘋果計算機」。
5. 點「加入」。

之後可直接從 iPhone 主畫面開啟使用。

如果圖示或功能沒有更新，請先刪除舊的主畫面圖示，再用 Safari 重新加入一次。

### Android 手機

1. 使用 Chrome 開啟計算機網址。
2. 點右上角 `⋮`。
3. 選擇「加到主畫面」或「安裝應用程式」。
4. 名稱輸入「小蘋果計算機」。
5. 點「新增」或「安裝」。

之後可直接從 Android 主畫面開啟使用。

## 十四、安裝到電腦

### Mac

1. 使用 Safari 開啟計算機網址。
2. 點上方選單「檔案」。
3. 選擇「加入 Dock」。
4. 名稱輸入「小蘋果計算機」。
5. 點「加入」。

完成後，計算機會出現在 Mac 的 Dock 與「應用程式」中。

### Windows

使用 Chrome：

1. 開啟計算機網址。
2. 點右上角 `⋮`。
3. 選擇「儲存及分享」或「更多工具」。
4. 選擇「建立捷徑」或「安裝頁面為應用程式」。
5. 勾選「以視窗開啟」。
6. 名稱輸入「小蘋果計算機」。
7. 點「建立」。

使用 Microsoft Edge：

1. 開啟計算機網址。
2. 點右上角 `…`。
3. 選擇「應用程式」。
4. 選擇「將此網站安裝為應用程式」。
5. 名稱輸入「小蘋果計算機」。
6. 點「安裝」。

## 十五、更新到 GitHub

如果只是在 Mac 本機修改，手機網址不會自動變成最新版。要讓手機、iPad、其他電腦看到最新版，需要把檔案上傳到 GitHub。

GitHub 專案網址：

`https://github.com/vanisa77-wq/xiaopingguo-calculator-codex`

GitHub Pages 使用網址：

`https://vanisa77-wq.github.io/xiaopingguo-calculator-codex/`

### 每次修改後要上傳的檔案

最重要：

- `index.html`：計算機正式版。功能或版面有改，一定要上傳。

如果有修改，也一起上傳：

- `app-icon.png`：手機或電腦圖示。
- `使用說明書.html`：快速說明書。
- `小蘋果計算機_使用說明書.md`：完整文字說明書。

### GitHub 上傳步驟

1. 打開 GitHub 專案。
2. 點「Add file」。
3. 選「Upload files」。
4. 把要更新的檔案拖進去。
5. 如果 GitHub 問是否取代同名檔案，保留新版即可。
6. 下方按「Commit changes」。
7. 等 1 到 3 分鐘，GitHub Pages 網址就會更新。

若手機仍看到舊版，請重新整理網頁，或刪除主畫面圖示後重新加入。
