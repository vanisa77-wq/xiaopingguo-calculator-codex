小蘋果計算機

快速使用說明

基本計算

  • 輸入數字與 ＋ − × ÷，再按 =。
  • 自動先乘除、後加減。
  • ( ) 會智慧開合括號。

清除與紙帶

  • ⌫ 單按刪一字；長按連續刪除。
  • AC 清除目前輸入，紙帶保留。
  • 紙帶可向上滑看歷史。

三種 ANS

  • ANS 0：四捨五入到整數。
  • ANS .00：四捨五入到小數第 2 位。
  • ANS ↑：無條件進位到整數。

設𝒳 記憶數字

  • 算完答案後按 設𝒳，把答案存成 𝒳。
  • 下一題可直接按 設𝒳 插入 𝒳。
  • 例：1+456788= → 設𝒳 → 123×𝒳=。

複製答案

  • 按 = 後會出現「複製」。
  • 可貼到 LINE、備忘錄或其他 App。
  • 按「截圖」會製作算式圖片並開啟分享選單。
  • 複製的數值也可當發票本金。

5% 稅金與發票

5%稅：直接輸入本金，或先複製本金後按鍵。

58,961 × 5% = 2,948　稅金

發票：可直接按，或先算稅金再按。

58,961 + 2,948 = 61,909　發票

稅額規則

5% 稅金以元為單位四捨五入，不保留小數；不是一律進位。

2,948.05 → 2,948

回到一般計算

稅金或發票完成後，直接按新數字即可開始下一題；標籤不會延續。

安裝與更新

iPhone：Safari 開網址 → 分享 → 加入主畫面。

Android：Chrome 開網址 → ⋮ → 加到主畫面。

Mac：Safari 開網址 → 檔案 → 加入 Dock。

Windows：Chrome/Edge 開網址 → ⋮ → 儲存/安裝為 App。
